# Diagnostic Detective Challenge

A dental case diagnosis game for PinkBlue.in's promotional campaign.

## Overview
Dentists diagnose real patient cases to earn rewards. The game is built with vanilla JS, Supabase, and Netlify Functions.

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```

2. Link to Netlify (if using CLI):
   ```bash
   netlify link
   ```

3. Run locally:
   ```bash
   npm start
   ```

## Structure
- `index.html`: Main entry point
- `styles/`: CSS modules
- `modules/`: JS logic modules
- `functions/`: Netlify serverless functions
- `database/`: SQL schema

## Deployment
Push to the `main` branch to deploy to Netlify.
Ensure Environment Variables are set in Netlify dashboard.
